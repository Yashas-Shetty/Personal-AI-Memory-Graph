This directory is reserved for the Next.js frontend of the Personal AI Memory Graph.

## Planned Tech Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS / Vanilla CSS
- **Interactions**: React Force Graph (for visualization)
- **State**: React Query / SWR (for API calls)

## How to get started
1.  Initialize a new Next.js app here: `npx create-next-app@latest ./`
2.  Configure the `.env.local` to point to `NEXT_PUBLIC_API_URL=http://localhost:8000`
